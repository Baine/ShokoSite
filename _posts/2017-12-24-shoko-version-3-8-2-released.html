--- id: 3409 title: Shoko Version 3.8.2 Released date: 2017-12-24T12:04:16+00:00 author: ElementalCrisis layout: post guid: https://shokoanime.com/?p=3409 permalink: /shoko-version-3-8-2-released/ image: assets/images/blog/2017/12/Shoko-Version-3.8.2-758x350.png categories: - Updates tags: - My Anime 3 - Shoko Desktop - Shoko Server --- Right in time for Christmas, Shoko Version 3.8.2 has been released! This release was mainly focused on fixing bugs discovered since the previous release and making the entire Shoko process smoother and easier to use. Utilities now load much quicker and random quirks with tags, seasons and filters have been fixed.  As always check the Changelog for a complete list of changes made. ## My Anime 3 Support For Version 3.8 First, thank you to all our My Anime 3 users for being patient while waiting for this release, more work was needed then we realized to get My Anime 3 working with Shoko Version 3.8 causing a lot of delays with this release. **You’ll need to first update to version 3.8.2 for it to work.**  Make sure to send your thanks to MaxPiva who spent many hours updating and fixing bugs in My Anime 3.  ## Download Link Download Shoko Version 3.8.2 ## Contributors This release was made possible by the following people.  **Avael, BigRetroMike, Cazzar, Da3dsoul, ElementalCrisis, MaxPiva, Netsplite.**
